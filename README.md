## ğŸš¦ Getting Started

### Prerequisites
```bash
node >= 18.0.0
npm or yarn
```

### Installation
```bash
# Clone the repository
git clone https://github.com/plagtech/mangoswap.git

# Navigate to project directory
cd mangoswap

# Install dependencies
npm install
```
